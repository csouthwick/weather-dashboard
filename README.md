# Weather Dashboard
A Weather Dashboard using the [OpenWeather API](https://openweathermap.org/api). Includes current weather, 5-day forecast, and a list of previously searched locations.

## Built With
* HTML
* CSS
* JavaScript
* jQuery
* Bootstrap
* Moment.js
